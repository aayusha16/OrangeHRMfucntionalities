**OrangeHRM Automation Framework**

This project contains automated test scripts for the OrangeHRM Demo Website using Python, Playwright, and Pytest.
The framework follows the Page Object Model (POM) design pattern to ensure better maintainability, scalability, and code reusability.It automates core features like Login, PIM (Add Employee), and Admin (Search User) modules. The framework uses a config file to avoid hardcoded values, markers for test categorization, and logging with screenshots on failures to improve maintainability, scalability, and debugging. Dependencies are listed in requirements.txt for easy setup.
